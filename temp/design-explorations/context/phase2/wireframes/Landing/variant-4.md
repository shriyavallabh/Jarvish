## Design Rationale

### Visual Identity
- **Dark background (#0A0E1A)** creates high-tech atmosphere
- **Neon accents** (cyan, purple, pink) add energy and modernity
- **Glowing effects** suggest AI intelligence and innovation
- **Playful micro-interactions** make finance tech feel approachable

### Trust Building Elements
- Tech-forward design appeals to digital-native advisors
- Live demo shows transparency and confidence
- Animated statistics create dynamic credibility
- Modern aesthetic differentiates from traditional platforms

### Typography Hierarchy
- **H1**: Space Grotesk 56px - Tech/gaming aesthetic
- **H2**: Space Grotesk 32px - Modern headings
- **Body**: Inter 16px - Clean readability on dark
- **Code/Stats**: JetBrains Mono - Technical precision

animations: {

## Accessibility Considerations

### WCAG AA Compliance
- **White on dark**: 15.5:1 contrast ratio ✓
- **Neon on dark**: Ensure 4.5:1 minimum
- **Focus states**: High contrast neon rings
- **Motion toggle**: Respect prefers-reduced-motion

### Motion Accessibility
```javascript
// Reduced motion fallbacks
@media (prefers-reduced-motion: reduce) {
  animations: {
    all: "none",
    transitions: "opacity 200ms ease"
  }
}
```

### Dark Mode Considerations
- Avoid pure black (#000) - use #0A0E1A
- Reduce brightness of neon colors for comfort
- Provide light mode toggle option
- Test with blue light filters

### Interactive Elements
- All cards keyboard navigable
- Focus trapping in demo area
- Clear hover/focus states
- Touch-friendly tap targets (48px min)

### Performance
- GPU-accelerated transforms only
- Lazy load animations below fold
- Use CSS animations over JavaScript
- Throttle scroll event listeners
